How to use

Simply fill all the stuff above the image, then when you're done, you can either save it like a normal image or copy paste it somewhere.
You can change color by typing [C:FFFFFF], with FFFFFF being an hexadecimal value for a color.
The text's style can be changed to italic with [T:I], to bold with [T:B], and can be reset to normal with [T:R].
A few icons are also available, add them with [I:P], [I:HEART] or [I:STAR].
